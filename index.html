Project Migration Notice

This project has been upgraded to a full-stack Next.js application!

The file index.html is no longer the entry point. Opening it directly in the browser will not work and caused the error you saw.

How to Run This Project:

  1. 1. Install Dependencies: 
    npm install
  2. 2. Set Up Environment Variables:

    Rename the file .env.local.example to .env.local and fill in your API keys.

  3. 3. Run the Development Server: 
    npm run dev
  4. 4. Open in Browser:

    Visit http://localhost:3000

The old React SPA files (index.tsx, App.tsx, etc.) are now obsolete and can be safely deleted.
